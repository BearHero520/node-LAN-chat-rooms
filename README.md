# node_sqlite3
一个简单的nodejs 服务 使用了express uuid sqlite3 可以局域网内聊天
默认访问端口 http://localhost:3333/



## 安装

>node 版本>=18.x

### 创建项目


```bash
$ npm install #  也可以  npm i  
$ npm run start
$ open http://localhost:3333 # 访问应用
$ npm run pkg # 资源构建
```
